O código em uma javascript:URL é avaliado como uma expressão, e o resultado ─ se não for undefined─ é usado (como uma string), como a URL do link neste caso. Mas aqui, a expressão de atribuição produz a string 'green', que não deveria ser um endereço de link; portanto, o voidoperador é necessário para descartar essa string, para que o navegador não saia da página quando o link for clicado.
